Trigonometrie & Winkel

Name

Beispiel
float a = 0.0;
float inc = TWO_PI/25.0;

for (int i = 0; i < 100; i=i+4) {
  line(i, 50, i, 50+sin(a)*40.0);
  a = a + inc;
}

Beschreibung

Generiert Zufallszahlen. Kann einen oder zwei Parameter annehmen. Wird nur high angegeben, werden Zahlen >= 0 und < high. Werden high und low angegeben, werden Zahlen >= low und < high generiert.

Berechnet den Abstand zwischen zwei Punkten

Berechnet den Winkel (im Bogenmaß) von einem bestimmten Punkt zum Koordinatenursprung, gemessen von der positiven x-Achse. Werte werden als Float im Bereich von PI bis -PI zurückgegeben. Die Funktion atan2 () wird am häufigsten verwendet, um die Geometrie an der Position des Cursors auszurichten.

Hinweis: Die y-Koordinate des Punkts ist der erste Parameter und die x-Koordinate ist der zweite Parameter, da die Berechnung der Tangente so strukturiert ist.

low: float untere Grenze		x1, y1, x2, y2: float Werte der Positioneny,x: float Werte
